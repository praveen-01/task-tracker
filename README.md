# Task Tracker CLI

A CLI Tool for efficiently managing day-to-day tasks. This tool allows users to add, update and manage tasks more easily. Developed this ClI as part of https://roadmap.sh/projects/task-tracker

## Installation

Install through pip
```
pip install git+https://github.com/praveen-01/task-tracker-cli.git
```

## Usage

### 1. Add a Task
```
 $ task-tracker-cli add "work on projects"
 $ task-tracker-cli list
+------+------------------+----------+---------------------+---------------------+
|   id | description      | Status   | Created At          | Updated At          |
+======+==================+==========+=====================+=====================+
|    1 | work on projects | To-do    | 2024-12-28 12:40:56 | 2024-12-28 12:40:56 |
+------+------------------+----------+---------------------+---------------------+
 
 $ task-tracker-cli add "walk 10k steps"
 $ task-tracker-cli list
+------+------------------+----------+---------------------+---------------------+
|   id | description      | Status   | Created At          | Updated At          |
+======+==================+==========+=====================+=====================+
|    1 | work on projects | To-do    | 2024-12-28 12:40:56 | 2024-12-28 12:40:56 |
+------+------------------+----------+---------------------+---------------------+
|    2 | walk 10k steps   | To-do    | 2024-12-28 12:42:07 | 2024-12-28 12:42:07 |
+------+------------------+----------+---------------------+---------------------+
```
### 2. Update description of an existing task with id
```
 $ task-tracker-cli update 1 "finish pending projects"
 $ task-tracker-cli list
+------+-------------------------+----------+---------------------+---------------------+
|   id | description             | Status   | Created At          | Updated At          |
+======+=========================+==========+=====================+=====================+
|    1 | finish pending projects | To-do    | 2024-12-28 12:40:56 | 2024-12-28 12:42:44 |
+------+-------------------------+----------+---------------------+---------------------+
|    2 | walk 10k steps          | To-do    | 2024-12-28 12:42:07 | 2024-12-28 12:42:07 |
+------+-------------------------+----------+---------------------+---------------------+
```
### 3. Modify status of a task with id
```
 $ task-tracker-cli modify 1 Done
 $ task-tracker-cli list
+------+-------------------------+-------------+---------------------+---------------------+
|   id | description             | Status      | Created At          | Updated At          |
+======+=========================+=============+=====================+=====================+
|    1 | finish pending projects | Done        | 2024-12-28 12:40:56 | 2024-12-28 12:43:11 |
+------+-------------------------+-------------+---------------------+---------------------+
|    2 | walk 10k steps          | In-progress | 2024-12-28 12:42:07 | 2024-12-28 12:43:02 |
+------+-------------------------+-------------+---------------------+---------------------+
```
### 4. delete a existing task with id
```
 $ task-tracker-cli delete 2
 $ task-tracker-cli list
+------+-------------------------+----------+---------------------+---------------------+
|   id | description             | Status   | Created At          | Updated At          |
+======+=========================+==========+=====================+=====================+
|    1 | finish pending projects | Done     | 2024-12-28 12:40:56 | 2024-12-28 12:43:11 |
+------+-------------------------+----------+---------------------+---------------------+
```
